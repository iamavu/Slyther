> Slyther is AWS Security tool to check read/write/delete access for S3 buckets

## Requirements 
```sh
aws-cli
```

## Installation

```sh
pip3 install -r requirements.txt
```

## Usage example

```sh
python3 slyther.py -b flaws.cloud
```
